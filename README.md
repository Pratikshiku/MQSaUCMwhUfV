### 三、功能描述
系统分为（身份）：普通用户、顾问、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 论坛
5. 公告
6. 健康知识
7. 个人中心
8. 预约顾问





**顾问功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 个人信息
6. 健康知识管理
7. 健康知识评论管理
8. 用户咨询管理
9. 论坛管理
10. 公告管理
11. 轮播图管理






**管理员功能：**
